# Laravel Certificate Authentication

This package provides certificate-based authentication for Laravel applications.

> **Requires [PHP 8.4+](https://php.net/releases/)**

## Installation

You can install the package via composer:

```bash
composer require a2insights/laravel-cert-auth
```

## Usage

```php
use A2Insights\LaravelCertAuth\Example;

$example = new Example();
echo $example->foo();
```

## Testing

```bash
composer test
```
